# BioTIMEr: Tools to Use and Explore the 'BioTIME' Database

The BioTIMEr package is developed in GitHub
(https://github.com/bioTIMEHub/BioTIMEr). To see the preferable citation
of the package, type citation("BioTIMEr"). The associated vignette
includes an introduction to the functions, how to apply them as well as
some hints on how to quantify and visualise temporal biodiversity change
for a given BioTIME dataset.
